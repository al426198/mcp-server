    Function registerGenerateCardPageTool

    • registerGenerateCardPageTool(server: McpServer): void

      HU202: Generación de páginas de tipo Card en lenguaje AL

      Genera una página de tipo Card en lenguaje AL.

      • id - ID del objeto.
      • name - Nombre del objeto.
      • sourceTable - Tabla de origen.
      • properties - Propiedades clave-valor de la página (opcional).
      • groups - Grupos de controles de la página.
      • actions - Acciones de la página (opcional).

      Parameters

      • server: McpServer

      Returns void

      La página AL generada.

      Example

      {
        "id": 50100,
        "name": "Cliente",
        "sourceTable": "Customer",
        "properties": {
          "ApplicationArea": "All",
          "Caption": "Ficha Cliente"
        },
        "groups": [
          {
            "name": "General",
            "fields": [
              {
                "name": "Nº Cliente",
                "sourceField": "No."
              },
              {
                "name": "Nombre",
                "sourceField": "Name"
              }
            ]
          }
        ],
        "parts": [
          {
            "name": "Foto",
            "source": "Customer Picture"
          }
        ],
        "areas": [
          {
            "name": "Processing",
            "type": "area",
            "actions": [
             {
                "name": "New",
                "properties": {
                  "Caption": "'Nuevo pedido'",
                  "Image": "NewDocument"
                }
              }
            ],
           "groups": [
            {
                "name": "RelatedInformation",

                 {
                    "name": "New",

                      "Caption": "'Nuevo pedido'",
                      "Image": "NewDocument"
                    }
                  }
                ]
              }
           ]
          }
        ]
      }
